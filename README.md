## Local Development

To run this blog locally:

1. Install Jekyll and dependencies:
```bash
gem install jekyll bundler
bundle install
```

2. Start the Jekyll server:
```bash
bundle exec jekyll serve
```

3. Visit `http://localhost:4000` in your browser
